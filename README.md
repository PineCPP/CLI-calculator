- What is this project?

A CLI (Command Line Interface) calculator.

- How to install the project?

Download the cpp file and build it according to your compiler, if you don't have one, download the prebuilt EXE file [THEN]
Create a new folder (anywhere) to store the exe file in [THEN]
Add the folder to the PATH variable

- How to use the project?

You must take into account what the name of the built exe file is, for the prebuilt version it's called "calculator.exe"
In CMD, type in the name of the file and you will see a menu showing you the format
